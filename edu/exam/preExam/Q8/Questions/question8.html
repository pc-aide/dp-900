Question 8

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

A Microsoft Power BI dashboard is associated with a single workspace:



A Microsoft Power BI dashboard can only display visualizations from a single dataset:



A Microsoft Power BI dashboard can display visualizations from a Microsoft Excel workbook:



Explanation:

A Microsoft Power BI dashboard is associated with a single workspace. A dashboard can display visualizations from multiple datasets and can also display visualizations from Microsoft Excel workbooks.

References:

  • Microsoft Power BI - Datasets Across Workspaces
  • Microsoft Power BI - End User Dashboards
  • Microsoft Power BI - Excel and Power BI
  • Microsoft Power BI - Workspaces

Correct Answers:
Statement 1: Yes
Statement 2: No
Statement 3: Yes
