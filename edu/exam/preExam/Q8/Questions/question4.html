Question 4

You need to develop a solution to provide data to executives. The solution must provide an interactive graphical interface, depict various key performance indicators, and support data exploration by using drill down. What should you use in Microsoft Power BI?






Explanation:

A report in Microsoft Power BI provides an interactive graphical interface with various key performance indicators and supports data exploration through features like drill down.

References:

  • Dashboards versus Reports in Power BI
  • Power BI Visualization - KPI
  • End-User Drill Down in Power BI

Correct Answer: B. a report
