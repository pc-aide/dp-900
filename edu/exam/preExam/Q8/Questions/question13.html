Question 13

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

You can copy a dashboard between Microsoft Power BI workspaces:



A Microsoft Power BI dashboard can only display visualizations from a single dataset:



A Microsoft Power BI dashboard can display visualizations from a Microsoft Excel workbook:



Explanation:

You can copy reports spanning more than one workspace. However, you can NOT copy dashboards across TWO different workspaces, as the two workspaces are comprised of two different sets of data and environments.

  • 1 - No - You can duplicate a dashboard. The duplicate ends up in the same Power BI workspace. There is no current functionality that allows you to move reports from one workspace to another. The new dashboard is saved in the same workspace as the original. The question has plural workspaces.
  • 2 - It is false. A Microsoft Power BI dashboard can display visualizations from multiple datasets. A dashboard can aggregate data from different sources, allowing the user to combine various visualizations into a single dashboard

  • Microsoft Power BI - Copying Dashboards
  • Microsoft Power BI - Excel Integration

Correct Answers:
Statement 1: No
Statement 2: No
Statement 3: Yes
