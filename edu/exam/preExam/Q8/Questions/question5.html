Question 5

Which two Azure services can be used to provision Apache Spark clusters? Each correct answer presents a complete solution.






Explanation:

Azure HDInsight and Azure Databricks are the two Azure services that can be used to provision Apache Spark clusters.

Azure Synaps Analytics could be possible too

References:

  • A Beginner's Guide to Azure Databricks
  • Apache Spark Overview on Azure HDInsight

Correct Answers: B. Azure HDInsight, C. Azure Databricks
