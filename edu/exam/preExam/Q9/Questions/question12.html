Question 12

You have an on-premises Microsoft SQL Server database. You need to migrate the database to the cloud. The solution must meet the following requirements:

  • Minimize maintenance effort.
  • Support the Database Mail and Service Broker features.

What should you include in the solution?





Explanation:

Azure SQL Database does not support Service Broker and Database Mail. You need the Managed Instance.

References:

  • Azure SQL Database Features Comparison

Correct Answer:
C. Azure SQL Managed instance
