Question 20

You have data saved in the following format:

{
	"firstName" : "John",
	"lastName" : "Doe",
	"age" : "23",
	"hobbies" : [
		{"type" : "sports", "value" : "Golf"},
		{"type" : "Lesure", "value" : "Reading"},
		{"type" : "Lesure", "value" : "Guitar"}
	]
}

Which format was used?

Explanation:

JSON data is written as name/value pairs (aka key/value pairs). A name/value pair consists of a field name (in double quotes), followed by a colon, followed by a value. For example: "name":"John".

Correct Answer:

  • D. JSON
