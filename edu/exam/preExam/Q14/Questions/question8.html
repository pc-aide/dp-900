Question 8

SSELECT, INSERT, and UPDATE are examples of which type of SQL statement?

Explanation:

DML stands for Data Manipulation Language. It includes SQL statements that are used to manipulate and manage data in the database. This includes commands like SELECT, INSERT, UPDATE, and DELETE, which are used to perform CRUD (Create, Read, Update, Delete) operations on the data.

References:

  • Data Manipulation Language (DML) - Satori Cyber

Correct Answer:
C. Data Manipulation Language (DML)
