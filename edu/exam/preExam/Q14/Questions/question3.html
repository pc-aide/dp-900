Question 3

Match the job roles to the appropriate tasks.

To answer, drag the appropriate job role from the column on the left to its task on the right. Each role may be used once, more than once, or not at all.

Draggable Roles:

Data analyst
Data architect
Data engineer
Database administrator

Drop Zones:

Explanation:

A database administrator is responsible for creating and maintaining transactional databases. A data engineer creates data pipelines and performs data ingestion tasks. A data analyst creates visualizations and charts to interpret data.

Correct Answers:

  • Create a pipeline to transfer data from a database to a data warehouse: Data engineer
  • Create a transactional database: Database administrator
  • Create a chart that shows sales transactions by week: Data analyst
