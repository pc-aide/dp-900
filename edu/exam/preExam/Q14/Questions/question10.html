Question 10

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

The pgAdmin tool can be used to manage Azure Database for PostgreSQL:



Azure Database for MariaDB offers point-in-time restore for up to 365 days:



Azure Database for MySQL always requires SSL connections:

Explanation:

Yes - The pgAdmin tool can indeed be used to manage Azure Database for PostgreSQL.

No - Azure Database for MariaDB offers point-in-time restore, but the backup retention period can be configured up to 35 days, not 365 days.

Yes - By default, Azure Database for MySQL enforces SSL connections between your server and your client applications to protect against “man in the middle” attacks.

References:

  • Explore Provisioning and Deployment of Relational Database Offerings in Azure - Microsoft Learn

Correct Answers:
The pgAdmin tool can be used to manage Azure Database for PostgreSQL: Yes
Azure Database for MariaDB offers point-in-time restore for up to 365 days: No
Azure Database for MySQL always requires SSL connections: Yes
