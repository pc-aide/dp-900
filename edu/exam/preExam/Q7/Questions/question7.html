Question 7

Your company is designing an application that will write a high volume of JSON data and will have an application-defined schema. Which type of data store should you use?






Explanation:


Document databases are optimized for storing and querying structured or semi-structured data as JSON documents. They allow for flexible schema design, which is defined by the application rather than being enforced by the database itself.

Key/Value databases cannot query JSON objects effectively, requiring retrieval and processing, which means limited query capabilities.

References:

  • Document Databases Overview
  • Key/Value Stores Overview

Correct Answer:
C. document
