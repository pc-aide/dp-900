Question 12

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

Extract, transform, & load (ETL) can reduce the transfer of sensitive data to destination systems:


Extract, load, & transform (ELT) transforms data by using a compute resource independent of the source system & destination system:


Extract, load, & transform (ELT) minimizes the time it takes to copy large volumes of data to destination systems:


Explanation:

  • 1 - You can reduce transfer sensitive data with transformations on data source. Data transformation that takes place usually involves various operations, such as filtering.
  • 2 - ELT uses the compute resource from the destination system, not from a separate compute resource.
  • 3 - ELT minimizes time as transformations are realized on target data destination systems. ELT processes skip the data copy step present in ETL, which can be a time-consuming operation for large data sets.

References:

  • ETL vs ELT - Microsoft Docs
  • ETL vs ELT Differences - Striim

Correct Answers:
Statement 1: Yes
Statement 2: No
Statement 3: Yes
