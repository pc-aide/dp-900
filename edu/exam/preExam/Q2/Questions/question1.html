Question 01

Match the Azure Data Factory components to the appropriate descriptions. To answer, drag the appropriate component from the column on the left to its description on the right. Each component may be used once, more than once, or not at all.

Dataset
Linked service
Mapping data flow
Pipeline
A representation of data structures within data stores
The information used to connect to external resources
A logical grouping of activities that performs a unit of work & can be scheduled

Explanation:

1. Dataset: A data set (or dataset) is a collection of data. In the case of tabular data, a data set corresponds to one or more database tables, where every column of a table represents a particular variable, and each row corresponds to a given record of the data set in question

2. Linked service: The information used to connect to external resources.

3. Pipeline: A logical grouping of activities that performs a unit of work and can be scheduled.

References:

  • Linked Services Overview
  • Pipelines and Activities

Correct Order:
Description 1: Dataset
Description 2: Linked service
Description 3: Pipeline
