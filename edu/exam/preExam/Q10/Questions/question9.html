Question 9

Select the answer that correctly completes the sentence.

combines a scalable, hig-performance Microsoft SQL Server-based relational data warehouse with the flexibility of a data lake & open-source Apache Spark

Explanation:

Azure Synapse Analytics is an integrated analytics service that brings together big data and data warehousing. It combines the scalability and performance of a relational data warehouse based on Microsoft SQL Server with the flexibility and capabilities of a data lake, enabling the integration and analysis of large datasets. Additionally, it incorporates Apache Spark, an open-source distributed processing system, which allows for advanced analytics and big data processing within the same service.

  • Azure Synapse Analytics
