Question 3

GRANT, REVOKE, and DENY are examples of which type of SQL statement?

Explanation:

DCL is a syntax similar to a computer programming language used to control access to data stored in a database (authorization). In particular, it is a component of Structured Query Language (SQL). Data Control Language is one of the logical group in SQL Commands.

References:

  • Data Control Language - Wikipedia

Correct Answer:

  • B. Data Control Language (DCL)
