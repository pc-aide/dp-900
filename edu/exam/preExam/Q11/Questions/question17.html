Question 17

Match the terms to the appropriate descriptions.

To answer, drag the appropriate term from the column on the left to its description on the right. Each term may be used once, more than once, or not at all.

Draggable Terms:

Data warehouses
Dimension tables
ETL
Fact tables
OLAP

Drop Zones:

Explanation:

ETL - Stands for Extract, Transform, Load, and is used to copy data from source systems into a data warehouse.

Fact tables - Contain numeric values that can be analyzed, usually found in a data warehouse.

Data warehouses - A solution that stores data in a relational schema optimized for read operations and analytics.

References:

  • ETL Process
  • Fact Table
  • Data Warehousing

Correct Answers:

  • A solution that copies data from files & OLTP databases into an analytical processing system: ETL
  • Contains numeric values that can be analyzed: Fact tables
  • A solution that stores data in a relational schema that is optimized for read operations: Data warehouses
