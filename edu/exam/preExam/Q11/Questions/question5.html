Question 5

In a fully normalized database, how is data read and written for a single entity?





Explanation:

It depends on the data operation that needs to be performed. Data can be read from multiple tables and written to a single or multiple tables as well.

References:

  • Core Data Concepts - Microsoft Learn
  • Database Normalization Description - Microsoft Office
  • Database Normalization - Wikipedia
  • DBMS Normalization - Javatpoint

Correct Answer:
A. Data is read from multiple tables and written to multiple tables.
