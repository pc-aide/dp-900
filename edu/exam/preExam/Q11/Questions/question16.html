Question 16

A bank has a system that manages financial transactions. When transferring money between accounts, the system must never retrieve a value for the source account that reflects the balance before the transfer and a value for the destination account that reflects the balance after the transfer. Of which ACID semantic is this an example?




Explanation:

Isolation is one of the four ACID semantic properties of a database management system that ensures different transactions do not interfere with each other. The isolation property guarantees that each transaction is executed in an isolated manner as if it is the only transaction taking place. This prevents situations where one transaction's results are visible to another transaction in an inconsistent way.

References:

  • Transactional Data Processing

Correct Answer:
D. Isolation
