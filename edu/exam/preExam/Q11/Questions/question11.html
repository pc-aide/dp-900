Question 11

In Azure Table storage, each row in a table must be uniquely identified by which two components? Each correct answer presents part of the solution.




Explanation:

In Azure Table storage, each row must be uniquely identified by a combination of a partition key and a row key. Together, these keys ensure that each entity within a table is uniquely identifiable.

References:

  • Azure Table Storage Overview

Correct Answers: A. a partition key, C. a row key
