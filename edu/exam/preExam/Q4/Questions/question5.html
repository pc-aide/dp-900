You have an Azure SQL database that you access directly from the Internet. You recently changed the public IP address of your computer. After changing the IP address, you can no longer access the database. You can connect to other resources in Azure. What is a possible cause of the issue?





Explanation: When you change the public IP address of your computer, the new IP address needs to be updated in the Azure SQL database's firewall rules. If the new IP address is not allowed in the firewall rules, you will be unable to access the database, even though you can connect to other Azure resources.

Correct Answer: D. A database-level firewall
