Question 8

Which command-line tool can you use to query Azure SQL databases?




Explanation:

  • A. sqlcmd - The sqlcmd utility lets you enter Transact-SQL statements, system procedures, and script files at the command prompt, making it the correct tool for querying Azure SQL databases.
  • B. bcp - The bulk copy program utility is used for bulk copying data between an instance of Microsoft SQL Server and a data file in a user-specified format, not for querying databases directly.
  • C. azdata - This tool is used for managing and querying data on Azure Data services, including SQL Server on Azure Arc and SQL Managed Instance. However, `sqlcmd` is more specifically associated with querying Azure SQL databases.
  • D. Azure CLI - The Azure CLI is a cross-platform command-line tool used for managing Azure resources but is not primarily designed for directly querying SQL databases.

References:

  • SQL Tools Overview - Microsoft Docs
  • Connect to Azure SQL Database using sqlcmd - Microsoft Docs

Correct Answer:
A. sqlcmd
