When can you use an Azure Resource Manager template?





Explanation: Azure Resource Manager (ARM) templates are used to automate the deployment and configuration of Azure resources in a repeatable and consistent manner. By defining the desired resources and their configurations in a JSON document, you can easily deploy them across different environments or regions.

Correct Answer: A. To automate the creation of an interdependent group of Azure resources in a repeatable way
