Question 16

You are deploying a software as a service (SaaS) application that requires a relational database for Online Transaction Processing (OLTP). Which Azure service should you use to support the application?




Explanation:

Azure SQL Database is a relational database and a managed service suitable for Online Transaction Processing (OLTP). Other options are incorrect for the following reasons:

  • A. Azure Cosmos DB is a non-relational database.
  • B. Azure HDInsight is used for big data and not for OLTP.
  • D. Azure Synapse Analytics is designed for data warehousing and not OLTP.

References:

  • Azure Database Review: Your Guide for Database Assessment - NetApp

Correct Answer: C. Azure SQL Database
