Question 15

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

Azure Data Studio can be used to query an Azure SQL database from a device that runs macOS:


Microsoft SQL Server Management Studio (SSMS) enables users to create & use SQL notebooks:


Azure Data Studio can be used to restore a database:


Explanation:

  • 1 - Yes - Azure Data Studio is a cross-platform tool that works on Windows, macOS, and Linux. It can query Azure SQL databases from any of these platforms, including macOS.
  • 2 - No - SQL Server Management Studio (SSMS) does not support SQL notebooks. For notebooks, Azure Data Studio should be used.
  • 3 - Yes - Azure Data Studio includes functionality to restore databases, making it a versatile tool for database management.

Statement 1: Yes
