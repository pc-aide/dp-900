Question 3

Which file format has defined names and data types for each column and uses compressed columnar storage?





Explanation:

Apache Parquet is a columnar storage file format that is designed for efficient data storage and query processing. It stores data in a columnar format, where each column is stored separately, allowing for efficient compression and selective column reads. Parquet files also include metadata that defines the names and data types of each column, enabling schema evolution and efficient query execution. Parquet is widely used in big data processing frameworks like Apache Spark and Apache Hadoop for its performance and storage efficiency.
