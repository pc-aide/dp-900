Question 16

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

The Cool access tier is the default tier of an Azure Storage account:



Append blobs are optimized for random read & write operations:



Retrieving data from the Archive access tier takes hours to complete:

Explanation:

N - The default access tier for a new general-purpose v2 storage account is set to the hot tier by default.

N - Append blobs are optimized for append operations, not random read & write.

Y - Retrieving data from the Archive tier can take several hours to complete (rehydration process).

References:

  • Azure Blob Storage Archive Tier Overview
  • Understanding Blobs

Correct Answers:
The Cool access tier is the default tier of an Azure Storage account: No
Append blobs are optimized for random read & write operations: No
Retrieving data from the Archive access tier takes hours to complete: Yes
