Question 5

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

Data stored in the Hot access tier of Azure Blob storage is stored on high-performance storage:



The cool access tier in Azure Blob storage incurs higher storage costs compared to the Hot access tier:



The Archive access tier in Azure Blob storage has the highest latency of the three available tiers:

Explanation:

Y - Data stored in the Hot access tier of Azure Blob storage is stored on high-performance storage, optimized for frequent access.

N - The cool access tier has lower storage costs but incurs higher access costs and is optimized for less frequently accessed data.

Y - The Archive access tier has the lowest storage costs but the highest data retrieval costs and latency, ideal for data that is rarely accessed.

References:

  • Azure Blob Storage Access Tiers Overview

Correct Answers:
Data stored in the Hot access tier of Azure Blob storage is stored on high-performance storage: Yes
The cool access tier in Azure Blob storage incurs higher storage costs compared to the Hot access tier: No
The Archive access tier in Azure Blob storage has the highest latency of the three available tiers: Yes
