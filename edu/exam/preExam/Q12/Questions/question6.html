Question 6

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

OLTP relies on a database system where data storage is optimized for both read & write operations:



OLAP relies on a database system where data storage is optimized for both read & write operations:



Data warehouses are optimized for read operations:

Explanation:

Y - OLTP systems are designed to efficiently process and store transactions, as well as query transactional data. They handle a large number of short online transaction processing requests.

N - OLAP systems are optimized for querying and reporting large datasets, typically using less normalized data, making them highly efficient for reads but less efficient for writes.

Y - Data warehouses are optimized for read operations, designed to store large amounts of historical data for analysis, often using OLAP technologies.

Correct Answers:
OLTP relies on a database system where data storage is optimized for both read & write operations: Yes
OLAP relies on a database system where data storage is optimized for both read & write operations: No
