API enables the use of SELECT statements to retrieve documents form Azure Cosmos DB.

Explanation:

The Azure Cosmos DB API for NoSQL supports querying documents using SQL. This article provides a sample document and two sample SQL queries and results.

MongoDB API uses syntax such as db.families.find( { id: "AdressfieldFamily" }, { children: true } ). NoSQL API uses syntax with the SELECT statement, such as SELECT * FROM Products.
