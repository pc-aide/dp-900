Question 14

For each of the following statements, select Yes if the statement is true. Otherwise, select No.




Explanation:

1. Platform as a service (PaaS) database offerings in Azure require less setup & configuration effort than infrastructure as a service (IaaS) database offerings. PaaS includes middleware, development tools, and more, which simplifies management and reduces setup complexity.

2. PaaS database offerings in Azure do not provide end users with the ability to control and update the operating system version. This is managed by the cloud service provider.

3. Not all relational and non-relational PaaS database offerings in Azure can be paused to reduce costs. For example, Azure SQL Database currently does not have a pause option. Deleting the database and re-creating it later is a workaround.

References:

  • What is PaaS - Azure
  • Stop Azure SQL Database - Microsoft Tech Community

Correct Answers:
1. Yes
2. No
3. No
