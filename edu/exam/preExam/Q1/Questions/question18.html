Question 18

Match the Azure Data Lake Storage Gen2 terms to the appropriate levels in the hierarchy.

To answer, drag the appropriate data type from the column on the left to its service on the right. Each data type may be used once, more than once, or not at all.

Image files
Key/value pairs
Relationships between employees
Azure Blob storage
Azure Table storage
Azure Cosmos DB Gremlin API

Explanation:

1. Azure Blob storage is ideal for storing large amounts of unstructured data such as image files.

2. Azure Table storage is designed for key-value storage, making it suitable for key/value pairs.

3. Azure Cosmos DB Gremlin API is optimized for managing graph data, which is useful for modeling relationships between entities.
