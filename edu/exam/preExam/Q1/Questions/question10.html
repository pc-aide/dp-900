To complete the sentence, select the appropriate option in the answer area:

is an object associated with a table that sorts & stores the data rows in the table based on their key values

Explanation:

These are the columns included in the index definition. There can be only one clustered index per table, because the data rows themselves can be stored in only one order. Nonclustered indexes have a structure separate from the data rows. A nonclustered index contains the nonclustered index key values and each key value entry has a pointer to the data row that contains the key value.
