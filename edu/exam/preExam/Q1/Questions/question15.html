Question 15

Which statement is an example of Data Manipulation Language (DML)?

Explanation:

Data Manipulation Language (DML) affects the information stored in the database. DML statements are used to insert, update, and change the rows in the database. The INSERT statement is an example of DML as it is used to add new rows of data to a table.

References:

  • T-SQL Statements - Microsoft Docs

Correct Answer:
C. INSERT
