Which type of database contains nodes and edges?





Explanation: A graph database shows two types of information, nodes and edges. The edges specify the relationships between nodes. Both nodes and edges own properties that provide details about that node or edge. This is similar to columns in a table. The edges can also have a direction to indicate the relationship's nature.

Correct Answer: A. graph
