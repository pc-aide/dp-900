Question 6

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

A job that processes sales data once daily is an example of a batch workload:


A job that calculates a rolling average temperature reading is an example of a streaming workload:


A job that calculates average revenue per product for the last month is an example of a streaming workload:


Explanation:

  • 1 - Yes - Batch processing refers to processing large volumes of data in a batch, typically scheduled at specific intervals. Processing sales data once daily fits this definition.
  • 2 - Yes - Stream processing involves analyzing continuous data in real-time. Calculating a rolling average temperature reading is an example of processing a continuous stream of data.
  • 3 - No - Calculating average revenue per product for the last month uses past data and is thus considered batch processing, not streaming.

References:

  • Difference Between Batch Processing and Stream Processing - GeeksforGeeks
  • Stream Processing - Microsoft Docs

Correct Answers:
Statement 1: Yes
Statement 2: Yes
Statement 3: No
