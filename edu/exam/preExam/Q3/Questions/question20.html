Your company needs to implement a relational database in Azure. The solution must minimize ongoing maintenance. Which Azure service should you use?





Explanation: To minimize ongoing maintenance while using a relational database in Azure, you should choose a Platform as a Service (PaaS) offering. Azure SQL Database is a managed relational database service that reduces the overhead of database maintenance tasks such as backups, patching, and scaling.

Correct Answer: B. Azure SQL Database
