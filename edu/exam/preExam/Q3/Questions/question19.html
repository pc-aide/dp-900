Select the answer that correctly completes the sentence.

You have an inventory management database that contains the following table:

ProductName Quantity
Product1 100
Product2 129
Product3 176

Which statement should you use in a SQL query to change the inventory quantity of Product1 to 270?





Explanation: The UPDATE statement is the simplest way to modify existing data in a table. It is used to change the value in an existing row. While a MERGE statement can also perform updates, it is more complex and requires two datasets to merge. Since the task is to update a specific value in an existing row, UPDATE is the appropriate choice.

Correct Answer: C. UPDATE
