Select the answer that correctly completes the sentence.

Which property of a transactional workload guarantees that each transaction is treated as a single unit that either succeeds completely or fails completely?





Explanation: All operations must be successfully completed for the transaction to complete: atomicity. Atomicity means that each statement in a transaction (to read, write, update, or delete data) is treated as a single unit. Either the entire statement is executed, or none of it is executed. This property prevents data loss and corruption, for example, if your streaming data source fails mid-stream.

Correct Answer: A. atomicity
