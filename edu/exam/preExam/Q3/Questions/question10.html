Question 10

Select the answer that correctly completes the sentence.

A key/value data store is optimized for

Explanation:

A key/value store associates each data value with a unique key. Key/value stores are highly optimized for applications performing simple lookups, where the data can be quickly accessed using a key. However, they are less suitable if you need to enforce constraints or perform operations like table joins, which are typical in relational databases.

Correct Answer: simple lookups
