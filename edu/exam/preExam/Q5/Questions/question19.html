Question 19

Structured data where each row represents a single data entity uses which type of schema?





Explanation:

Tabular data refers to data that is organized into a table, where information is arranged in rows and columns. Each row typically represents a record or an observation, while each column represents a specific attribute or variable related to the records.

References:

  • Introduction to the Tabular Object Model
  • What is Tabular Data?

Correct Answer: B. tabular
