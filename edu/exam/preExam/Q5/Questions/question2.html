Question 2

A team of developers has computers that run Windows 10 and Ubuntu Desktop. The developers need to connect to and query an Azure SQL database from each of their computers. The developers require code assistance features such as IntelliSense. What should the developers use?




Explanation:

SSMS can run queries with intelisense, but it does not work on Ubuntu.

Azure Data Studio is a cross-platform database tool for data professionals who use on-premises and cloud data platforms on Windows, macOS, and Linux. Azure Data Studio offers a modern editor experience with IntelliSense, code snippets, source control integration, and an integrated terminal. SSMS doesn't run queries, it handles configuration and management.

Correct Answer: C. Azure Data Studio
