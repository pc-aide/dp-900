Question 5

What is a benefit of hosting a database on Azure SQL Managed Instance as compared to an Azure SQL Database?




Explanation:

Options A, C, and D are available for both Azure SQL Database and Azure SQL Managed Instance. However, native support for cross-database queries and transactions is a distinct advantage of Azure SQL Managed Instance.

References:

  • Azure SQL Database vs. SQL Managed Instance Features Comparison

Correct Answer: B. native support for cross-database queries and transactions
