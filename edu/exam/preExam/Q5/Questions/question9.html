A relational database must be used when:


Explanation:

1. Relational databases are static schema, so option A is wrong.

2. Data stored as key/value pairs is JSON (semi-structured data); relational databases are structured data. Option B is wrong.

3. Storing large images and videos is unstructured data. Option C is wrong.

You do not have to resort to a relational database if you want to enforce ACID. There are many tools that can enforce ACID in other NoSQL databases; it does not make relational databases the only way.

Relational databases are used for strong consistency guarantees, emphasizing the atomic nature of RDBMS.

Correct Answer: strong consistency guarantees are required
