Question 12

You need to query a table named Products in an Azure SQL database. Which three requirements must be met to query the table from the internet?





Explanation:

A, D: Resource group permissions are not required to query an Azure SQL database table.
B, C, E: Your IP must be allowed in the database firewall, and you need a database user with SELECT privileges on the desired table.

References:

  • Getting Started with Database Engine Permissions

Correct Answers:
B. You must have SELECT access to the Products table.
C. You must have a user in the database.
E. Your IP address must be allowed to connect to the database.
