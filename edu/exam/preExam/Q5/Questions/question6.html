Question 6

You need to design and model a database by using a graphical tool that supports project-oriented offline database development. What should you use?




Explanation:

SSDT, you can create an offline database project and implement schema changes by adding, modifying or deleting the definitions of objects (represented by scripts) in the project

References:

Correct Answer: A. Microsoft SQL Server Data Tools (SSDT)
