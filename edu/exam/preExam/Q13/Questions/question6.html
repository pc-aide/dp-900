Question 6

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

Database administrators apply data cleansing routines & turn data into useful information:



Data engineers manage databases, store backup copies of data, & restore data in the event of a failure:



Data analysts create data visuals & enable companies to make data-driven decisions:

Explanation:

No - Database administrators are responsible for the design, implementation, maintenance, and operational aspects of database systems. They're not primarily responsible for data cleansing routines.

No - Data engineers design and implement data-related workloads but are not responsible for managing backups or restoring data in the event of failure.

Yes - Data analysts create data visuals and enable companies to make data-driven decisions by exploring data and building reports.

References:

  • Explore Job Roles - Microsoft Learn

Correct Answers:
Database administrators apply data cleansing routines & turn data into useful information: No
Data engineers manage databases, store backup copies of data, & restore data in the event of a failure: No
Data analysts create data visuals & enable companies to make data-driven decisions: Yes
