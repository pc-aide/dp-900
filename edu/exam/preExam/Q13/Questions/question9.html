Question 9

Which language is used to define queries in Azure Synapse Data Explorer?






Explanation:

C. KQL - Data Explorer democratizes self-service, big data analytics with the intuitive Kusto Query Language (KQL) that provides the expressiveness and power of SQL with the simplicity of Excel. KQL is highly optimized for exploring raw telemetry and time-series data by leveraging Data Explorer's best-in-class text indexing technology for efficient free-text and regex search, and comprehensive parsing capabilities for querying trace data, text data, and semi-structured data like JSON.

References:

  • Azure Synapse Data Explorer Overview

Correct Answer:
C. KQL
