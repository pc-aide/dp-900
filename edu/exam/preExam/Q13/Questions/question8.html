Question 8

Which statement is an example of Data Manipulation Language (DML)?






Explanation:

A. INSERT - DML (Data Manipulation Language) statements allow you to query, edit, add, and remove data stored in database objects. The primary DML commands are SELECT, INSERT, DELETE, and UPDATE. Using DML statements, you can perform powerful actions on the actual data stored in your system.

References:

  • DML Overview - dbt Documentation

Correct Answer:
A. INSERT
