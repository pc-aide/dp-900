Question 3

Match the data processing languages to the appropriate real-time analytics services.

To answer, drag the appropriate language from the column on the left to its service on the right. Each language may be used once, more than once, or not at all.

Draggable Languages:

JavaScript
KQL
Python
SQL

Drop Zones:

Explanation:

Azure Data Explorer: KQL (Kusto Query Language) is used for analyzing large volumes of data in real-time.

Azure Stream Analytics: SQL is used to process high-velocity, real-time streaming data.

Azure Synapse Analytics: SQL is used to query and analyze structured data, and Python is used for data science tasks.
