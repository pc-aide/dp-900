Question 15

Match the Azure Cosmos DB APIs to the appropriate data structures.

To answer, drag the appropriate API from the column on the left to its data structure on the right. Each API may be used once, more than once, or not at all.

Draggable APIs:

Azure Cosmos DB for Apache Cassandra API
Azure Cosmos DB for Apache Gremlin API
Azure Cosmos DB for NoSQL API
Azure Cosmos DB for Table API

Drop Zones:

Explanation:

Azure Cosmos DB for Apache Gremlin API is used for graph data. Azure Cosmos DB for NoSQL API handles JSON documents. Azure Cosmos DB for Table API is used for key/value data.

References:

  • Querying in Azure Cosmos DB
  • Azure Cosmos DB Gremlin API
  • Azure Cosmos DB Table API

Correct Answers:

  • Graph data: Azure Cosmos DB for Apache Gremlin API
  • JSON documents: Azure Cosmos DB for NoSQL API
  • Key/value data: Azure Cosmos DB for Table API
