To complete the sentence, select the appropriate option in the answer area.

When provisioning an Azure Cosmos DB
, you need to specify which type of API you will use.

Explanation:

When provisioning an Azure Cosmos DB, you need to specify the type of API you will use at the Account level. The choice of API is an initial step during the creation of the Account, as each API has specific features that influence the structure and management of data within the Cosmos DB account.

Correct Answer: account
