You can query a graph database in Azure Cosmos DB

Explanation:

Gremlin language is not a SQL-like language. It is based on the Gremlin API. Example: g.V().hasLabel('person').has('age', gt(40)) --- filter query.

I work with Cosmos DB and Gremlin every day. You CAN access the underlying documents with the SQL API, but you can only query by nodes and edges with the Gremlin API.
