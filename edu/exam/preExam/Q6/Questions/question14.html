Question 14

What is a benefit of the Azure Cosmos DB Table API as compared to Azure Table storage?




Explanation:

Multi-master support for Azure Cosmos DB is now available in all public regions. Azure Cosmos DB Table API is a key-value storage hosted in the cloud. It's a part of Azure Cosmos DB, that is Microsoft's multi-model database.
