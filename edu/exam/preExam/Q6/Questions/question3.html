Question 3

Which setting can only be configured during the creation of an Azure Cosmos DB account?





Explanation:

You must create a separate account for each API. The API cannot be changed after creation. Other settings such as geo-redundancy and multi-region writes can be modified after the account is created.

API is fixed during the creation process of the Cosmos DB account.

References:

  • Managing Cosmos DB Accounts

Correct Answer: API
