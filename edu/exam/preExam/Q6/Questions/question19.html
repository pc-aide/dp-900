Question 19

You need to store data by using Azure Table storage. What should you create first?




Explanation:

First create an Azure storage account, then use Table service in the Azure portal to create a table. An Azure storage account contains all of your Azure Storage data objects: blobs, files, queues, and tables.

References:

  • Microsoft - Azure Table Storage Quickstart
  • Microsoft - Create a Storage Account

Correct Answer: B. a storage account
