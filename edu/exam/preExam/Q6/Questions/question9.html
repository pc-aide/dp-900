Question 9

Which type of non-relational data store supports a flexible schema, stores data as JSON files, and stores all the data for an entity in the same document?




Explanation:

The key is JSON = document. Document-oriented databases store data as JSON files and support a flexible schema, which allows all the data for an entity to be stored in a single document.

  • Data Store Overview
  • Document-oriented Database

Correct Answer: A. Document
